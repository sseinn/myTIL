# #define _CRT_SECURE_NO_WARNINGS 

- **CRT** : CRT란 윈도우에서 scanf와 같은 기능을 제공하는 C 런타임 라이브러리다.

- Visual Studio는 scanf()의 더 안전한 버전인 scanf_S()를 사용하라고 경고한다.

- scanf()를 사용하고 싶으면 선행처리 정의를 넣으면 경고를 없앨 수 있다.
