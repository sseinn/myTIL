# p.25 예제

```
#include <stdio.h> //헤더 부분

int main(void)
{
	/* 변수 선언 */
	int i; 
	int j;
	int k;

	/* 계산 부분 */
	i = 5;
	j = 9;
	k = i * j;

	/* 출력 부분 */
	printf("%d * %d = %d\n", i, j, k);

	return 0;
} //메인 함수 끝
```

- 출력 결과

  5 * 9 = 45
