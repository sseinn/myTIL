# p.228 예제
```
#include <stdio.h>
double square(double x)
{
	double number;
	number = (x - 1) * (x - 1);
	return number;
}

int main(void)
{
	double num1, num2;
	printf("숫자를 입력하시오: ");
	scanf_s("%lf", &num1);
	num2 = square(num1);
	printf("입력값: %lf, 출력값: %lf\n", num1, num2);
	return 0;
}
```

- 실행 결과
```
숫자를 입력하시오: 5
입력값: 5.000000, 출력값: 16.000000
```

# p.230 예제
```
#include <stdio.h>
#define WON_DOLLAR 1200
float exchange(float money)
{
	float ex;
	ex = money * WON_DOLLAR;
	return ex;
}

int main(void)
{
	float won, dollar;
	printf("달라 금액을 입력하시오: ");
	scanf_s("%f", &dollar);
	won = exchange(dollar);
	printf("%f 달라는 %f 원입니다.\n", dollar, won);
	return 0;
}
```

- 실행 결과
```
달라 금액을 입력하시오: 100
100.000000 달라는 120000.000000 원입니다.
```
